برمجة الكائنات في بايتون

برمجة الكائنات هي نموذج برمجي يستخدم الكائنات (objects) لتنظيم وتحليل البيانات. في بايتون، تعد البرمجة الكائنية وسيلة قوية لتنظيم الشيفرات وجعلها أكثر مرونة وإعادة استخدام.

المفاهيم الأساسية

  • الفصول (Classes): هي القوالب التي يتم إنشاء الكائنات بناءً عليها.
  • الكائنات (Objects): هي مثيلات للفصول وتحتوي على بيانات وسلوكيات.
  • الوراثة (Inheritance): هي ميزة تسمح للفصول بترث خصائص وسلوكيات من فصول أخرى.
  • تعدد الأشكال (Polymorphism): هو القدرة على استخدام نفس الاسم لأشياء مختلفة بناءً على السياق.

مثال على برمجة الكائنات

فيما يلي مثال بسيط يوضح كيفية استخدام البرمجة الكائنية في بايتون:

class Animal:
    def __init__(self, name):
        self.name = name

    def speak(self):
        return "Animal sound"

class Dog(Animal):
    def speak(self):
        return "Woof!"

# إنشاء كائن من الفصل Dog
my_dog = Dog("Buddy")
print(my_dog.name)  # إخراج: Buddy
print(my_dog.speak())  # إخراج: Woof!

في هذا المثال، قمنا بإنشاء فصل `Animal` وفصل `Dog` الذي يرث من `Animal`. ثم قمنا بإنشاء كائن من فصل `Dog` واستخدامه.
